## Installation

Clone this repository

```bash
git clone https://github.com/MuhammadSdk/muhammad-sidik-portfolio.git
```

Go to project folder

```bash
cd muhammad-sidik-portfolio
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```
